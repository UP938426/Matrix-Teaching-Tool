2x2 Matrix Addition and Subtraction

Matrices of the same size can be added to and subtracted from one another. The addition and subtraction are executed element-by-element. The final matrix has the same dimensions as the matrices involved.


If we have Matrices A and B


A      B
2 4
3 7
1 5
0 7

For Addition we would use the following maths.

2+1 4+5
3+0 7+7

Giving us the result

3 9
3 14

For Subtraction we would use the following maths.

2-1 4-5
3-0 7-7

Giving us the result

1 -1
3 0
