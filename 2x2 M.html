2x2 Matrix Multilpication

Matrices of the same size can be multiplie by one another or by a scalar (real number). By simply multiplying all its elements by this scalar. The final matrix has the same dimensions as the initial matrix.


We will use Matrices A and B as examples.

Multiplication becomes slightly more complicated when a matrix is multiplied by another matrix. We will now look into the required method in a 2x2 senerio, but the same techniques ring true with 3x3, 3x4, 4x3 etc as well. For the product AB to exist, the numer of columns of A must equal the number of rows of B.

Matrix A is size M x N, and matrix B is size K x L. Therefore N must equal K. The resulting Matrix C will have the dimensions M x L. Below is an example of the rule.

A 1 2
1 a 11 a 12
2 a 21 a 22
B 1 2
1 b 11 b 12
2 b 21 b 22
(A X B) 1 2
1 (a 11*b 11) + (a 12*b 21) (a 11*b 12) + (a 12*b 22)
2 (a 21*b 11) + (a 22*b 21) (a 21*b 12) + (a 22*b 22)
