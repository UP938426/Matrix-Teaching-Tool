Types of Matrices and Terminology

Square matrix → Will have the same number of rows and columns.
Unit Matrix → A square matrix having ones along its main diagonal and zeroes in all other postions.
Null Matrix → A matrix where every element is a zero.
Main diagonal → the Diagonal running from the upper-left element to the lower-right element.
Secondary Diagonal → the Diagonal from upper-right to lower-left.

Transposing a Matrix.
This is where swaps the rows and colums of the matrix, turning a 3x2 matrix into a 2x3 Matrix for example.

1 9 0
-4 5 2

↓

1 -4
9 5
0 2

Identity Matrices

The Identtity Matrix is the equivalent of the number one. It can be 2x2, 3x3, 4x4 and so on. Below is the Identity matrix of 2x2 and 3x3.

1 0
0 1

1 0 0
0 1 0
0 0 1
